Instructions!

Aim Of The Game!

The aim of the game is simple, match tiles of the same number to add them together. To win the game, you need a tile with 2048.

How To Play!

Shift all the tiles in a chosen direction, when two tiles with the same number are pushed into each other, they get added together. After every move, a new tile will be created in a random empty location. The game ends when either when you get the winner score, or when no more moves are possible.

Controls!

Moving Tiles: Arrow Buttons, Arrow Keys or "W,A,S,D" keys.


Navigating Menus: Click/Tap on the button to open. Click/Tap on the button, or press "ESC" to close.
